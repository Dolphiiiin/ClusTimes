cluster 新聞部 ルールとガイドライン

cluster 新聞部活動の方向性

cluster 新聞部は、記事を書く人、広告を作る人が、ゆるくたのしく続けられるように活動します

ClusTimesの目指すもの

利用機器にかかわらずclusterに新しく来た人にもわかりやすく楽しめる新聞づくりを目指します。このため、cluster内外のさまざまな人やコミュニティ、イベントの情報をとりあげるよう努めます。

記事・広告の寄稿の仕方

このルールに賛同するひとは、だれでも記事や広告を寄稿することができます。寄稿方法やフォーマットなどのサポートは「記者」のロールをもった人が担当します。サポートの担当する人は「副編集長」または「編集長」が割り当てますので、Discordの雑談チャンネルなどでお気軽にやりたいことを書いてください。

記事・広告の掲載判断

ClusTimesでは、記事・広告の寄稿を歓迎していますが、誌面の量やこれまでの号の掲載記事などを考えてさまざまな情報をとりあげるように選択します。誌面の掲載については編集部キャップが立案し、編集長または副編集長が承認して決定します。
記事や広告を寄稿していただいても掲載できない場合もあることご了承ください。

記事・広告の著作権について

cluster 新聞部はCreative Commons「表示 4.0 国際 (CC BY 4.0)」のライセンスで発行します。記事・広告の寄稿は「表示 4.0 国際 (CC BY 4.0)」のライセンスでの発行に合意したものとします。

cluster 新聞部のルール

上記の項目に同意することにより、cluster 新聞部のDiscordサーバーに参加し、また、記事・広告を寄稿することができます。このルールは、cluster 新聞部のDiscordサーバーで告知を経た後、編集長または副編集長の承認により随時更新されます。
